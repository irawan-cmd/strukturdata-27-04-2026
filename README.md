# strukturdata-27-04-2026

🎨 Queue Visualizer — Python GUI

Aplikasi visualisasi Struktur Data Queue berbasis Python dengan tampilan modern menggunakan CustomTkinter.

Project ini menggabungkan 5 studi kasus Queue dalam satu aplikasi interaktif.

🚀 Fitur
🖨️ Printer Queue (FIFO) — simulasi antrian dokumen
🥔 Hot Potato (Circular Queue) — rotasi & eliminasi pemain
🏥 Rumah Sakit (Priority Queue) — pasien berdasarkan prioritas
🌐 BFS Traversal — pencarian graf menggunakan queue
✈️ Bandara — simulasi antrian penumpang

🛠️ Teknologi
- Python 3
- CustomTkinter
- collections.deque
